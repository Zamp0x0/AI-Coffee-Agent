☕ AI咖啡 Agent
===============================

🤖 AI咖啡 Agent 是一个基于大模型的智能咖啡点单与客服系统。
用户可以用自然语言查询菜单、获取推荐，并直接让 AI 帮你下单和预约。
系统将「对话 + 知识库 + 订单系统」打通，实现真正能干活的 AI Agent。

📌 同时支持 RAG 向量检索、函数调用下单与多用户对话记忆。

--------------------------------------------------
🚀 项目功能
--------------------------------------------------

AI咖啡 Agent 支持：

☕ 咖啡与菜单查询  
🧠 智能推荐（根据口味、苦度、价格等）  
🛒 自然语言下单  
📅 预约与订单管理  
📚 知识库问答（基于 Redis 向量库）  

用户可以直接说：

“美式有吗？”  
“帮我点一杯拿铁”  
“推荐一杯不苦的咖啡”  

AI 会自动调用后台服务并返回真实结果。

--------------------------------------------------
🧱 技术栈
--------------------------------------------------

🧩 JDK：17  
🧩 后端：Spring Boot 3.5.3  
🧩 大模型：Qwen（通义千问）  
🧩 数据库：MySQL 8.0  
🧩 知识库：Spring AI RAG + Redis Stack  
🧩 前端：Vue 3  

--------------------------------------------------
🧠 系统架构
--------------------------------------------------

用户输入  
   → AI咖啡 Agent（Spring AI）  
       → 大模型 Qwen  
           → RAG 向量检索（Redis）  
               → 咖啡 / 订单 / 商户 API  
                   → MySQL  

--------------------------------------------------
⚙️ 运行方式
--------------------------------------------------

1️⃣ 启动依赖服务  
sh docker/run.sh  

2️⃣ 启动 order-server 服务  

3️⃣ 启动 AI 客服服务 
- 选择 .env 作为环境变量  
- 看不到 .env：  
  - macOS：Command + Shift + .  
  - Windows：Ctrl + H  

4️⃣ 启动前端  
npm install  
npm run dev
